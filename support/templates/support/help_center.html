Module Guides

Learn how each module works and what to avoid

How It Works

3-Panel Layout:

  • Left Panel (Groups): Shows all available item groups
  • Middle Panel (Items): Shows items with checkboxes
  • Right Panel (Preview): Your selected items with Sl.No, Description, Unit, Qty, Rate, Amount
  1. Select Work Type (Original/Repair) and Category
  2. Click a Group to load its items
  3. Check items you need
  4. Enter Quantity for each item
  5. Drag items to reorder
  6. Click Download Estimate
DON'Ts
  • Don't rapidly click multiple checkboxes
  • Don't leave Quantity empty
  • Don't refresh while items load

How It Works
  1. Prepare an Excel file with your item data
  2. Upload the file
  3. System reads and processes your data
  4. Download the formatted estimate output
DON'Ts
  • Don't upload files larger than 10MB
  • Don't upload non .xlsx/.xls formats
  • Don't have empty rows in your data
  • Don't merge cells

How It Works

Purpose: Compare estimated vs actual executed quantities.

  1. Upload your estimate file first
  2. Enter Executed Quantity for each item
  3. Set T.P % and type (Excess/Less)
  4. Optionally Add Supplemental Items
  5. Click Download Workslip
DON'Ts
  • Don't forget T.P % if applicable
  • Don't click Clear All unless intended
  • Don't expect supplemental items unless you CHECK them

How It Works

Purpose: Calculate costs where pricing is Qty x Days x Rate.

  1. Select Category (Temp Electrical / Temp Civil)
  2. Select a Group
  3. Click Add next to items you need
  4. Enter Qty and Days
  5. Click Download
DON'Ts
  • Don't enter Days as 0
  • Don't leave Qty empty
  • Don't add the same item twice

4 Bill Types
  1. Bill from Estimate: CC First & Part / First & Final
  2. Nth to Nth Bill: 2nd, 3rd... running bills
  3. Part to Final Bill: Final from part bill
  4. Nth to Final Bill: Final from last running bill
DON'Ts
  • Don't upload Excel files for templates - only Word (.docx)
  • Don't skip MB details
  • Don't confuse Part Bill with Nth Bill

How It Works
  1. Create a Word/Excel file with placeholders like {{ "{{NAME}}" }}
  2. Upload and name your template
  3. Select template, fill in values
  4. Download filled document
DON'Ts
  • Don't use spaces in placeholders - use underscores
  • Don't split placeholders across lines
  • Placeholders are case-sensitive

How It Works

Same 3-Panel layout as New Estimate, but uses AMC-specific backend data.

DON'Ts
