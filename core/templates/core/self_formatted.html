How Self Formatted Forms Work

1
The Concept
You have a source file (Forwarding Slip, Estimate, Bill, etc.) that contains data like work name, amount, contractor name, MB details, etc.
You have a template file (Word/Excel) where you want this data to appear — but in your own format.
You place {% verbatim %}{{PLACEHOLDERS}}{% endverbatim %} in the template wherever you want data to appear. The system reads the source, extracts values, and fills your template automatically.
2
Your Source File (Input)
This is your existing file — a forwarding slip, estimate, or bill Excel/Word/PDF. For example:
Source File — Forwarding_Slip.xlsx
Name of the Work : Construction of CC Road at Ward No.5
Name of the Agency : Sri. Ramesh Kumar
Agreement No : AGR/2024-25/156
Administrative Sanction : Proc.No.123/2024, dt: 15-03-2024
Technical Sanction : TS/456/2024, dt: 20-03-2024
M.B No : MB-45, Pages 12 to 18
Amount : Rs. 4,85,000.00
Tender Premium : 15% Excess
3
Your Template File (with Placeholders) → Filled Output
In your template, put {% verbatim %}{{PLACEHOLDER_NAME}}{% endverbatim %} wherever data should go. Click the Placeholders button to see all available names and copy them.
Template — My_Bill_Format.docx
OFFICE OF THE EXECUTIVE ENGINEER

Sub: {% verbatim %}{{NAME_OF_WORK}}{% endverbatim %}

Ref: Agreement No. {% verbatim %}{{REF_OF_AGREEMENT}}{% endverbatim %}

Contractor: {% verbatim %}{{CONTRACTOR}}{% endverbatim %}

A.S: {% verbatim %}{{ADMIN_SANCTION}}{% endverbatim %}
T.S: {% verbatim %}{{TECH_SANCTION}}{% endverbatim %}

Bill Amount: Rs. {% verbatim %}{{AMOUNT}}{% endverbatim %}
In Words: {% verbatim %}{{AMOUNT_IN_WORDS}}{% endverbatim %}

M.B Details: {% verbatim %}{{MB_DETAILS}}{% endverbatim %}
T.P: {% verbatim %}{{TP_DETAILS}}{% endverbatim %}
→
Output — Filled_Document.docx
OFFICE OF THE EXECUTIVE ENGINEER

Sub: Construction of CC Road at Ward No.5

Ref: Agreement No. AGR/2024-25/156

In Words: Rupees Four Lakhs Eighty Five Thousand Only

T.P: 15% Excess

5
Custom Placeholders
Need a field that's not in the built-in list? Create a Custom Placeholder:
Custom Placeholder Example
Placeholder Name Label to Find What Happens
DIVISION_NAME Name of Division System searches source file for "Name of Division" and extracts the value next to it
CIRCLE_NAME Name of Circle System finds "Name of Circle" in source and extracts the value

Then in your template, use {% verbatim %}{{DIVISION_NAME}}{% endverbatim %} and {% verbatim %}{{CIRCLE_NAME}}{% endverbatim %} — they'll be replaced with the found values.

6
Quick Steps
Prepare Template
Open your Word/Excel template. Type {% verbatim %}{{PLACEHOLDER}}{% endverbatim %} wherever data should go. Use the Placeholders button to copy names.
Upload Files
Upload your source file (forwarding slip/estimate/bill) and your template file here.
Generate
Click Preview to check extracted values, then Generate to download the filled document!
Tips: Use Save Reusable Format (right side) to save your template so you only upload the source file next time. The Preview button lets you verify extracted values before generating. Supported source files: Excel, Word, PDF (including scanned), Images.
